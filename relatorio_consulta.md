## Arquitetura
O repositório janiojandson/consulta tem a seguinte estrutura:

- .github (dir)
- .gitignore (file)
- .port_sessions (dir)
- CLAW.md (file)
- PARITY.md (file)
- README.md (file)
- assets (dir)
- rust (dir)
- src (dir)
- tests (dir)

## Funcionalidades Principais
O sistema de consultas opera da seguinte forma:

*   O diretório `src` contém o código fonte do projeto.
*   O diretório `tests` contém os testes unitários e de integração do projeto.
*   O diretório `rust` contém o código fonte do projeto em Rust.
*   O arquivo `README.md` contém as instruções de uso do projeto.
*   O arquivo `CLAW.md` contém as instruções de uso do projeto CLAW.
*   O arquivo `PARITY.md` contém as instruções de uso do projeto PARITY.
