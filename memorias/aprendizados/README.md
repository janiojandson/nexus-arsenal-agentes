# 💡 Aprendizados do Nexus

Insights, erros documentados e soluções que o Nexus descobre durante a operação.

## Regras
1. Todo erro que causou retry ou fallback DEVE ser documentado aqui.
2. Toda solução que funcionou DEVE ser registrada para referência futura.
3. Antes de executar qualquer tarefa complexa, consultar este diretório.
4. **Nunca repetir um erro já documentado.**

## Formato Sugerido
```markdown
## [Data] — [Título do Aprendizado]
- **Problema:** [descrição]
- **Solução:** [o que funcionou]
- **Contexto:** [quando ocorreu, qual modelo/provider]
- **Lição:** [resumo em 1 frase]
```
