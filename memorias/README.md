# 🧠 Memórias do Nexus

Este diretório é o **banco de dados principal** do Cérebro Nexus.
Toda memória persistente, aprendizados, insights e registros de tarefas vivem aqui em formato Markdown ou JSON.

## Estrutura

- `diario_de_bordo.md` — Diário de Bordo Vivo (registro cronológico de cada ciclo)
- `alertas/` — Alertas de instabilidade de modelos (Watchdog)
- `aprendizados/` — Insights, erros documentados e soluções
- `tarefas/` — Registros de tarefas executadas e seus resultados

## Regras

1. **SQL é apenas para estados temporários** (ex: evitar responder 2x a mesma msg no WhatsApp).
2. **GitHub é a fonte de verdade** para memória de longo prazo.
3. **Busca de memória pré-ação**: Antes de qualquer tarefa complexa, consultar este diretório.
4. **Nunca repetir erros**: Se algo falhou e foi documentado aqui, não repetir.
